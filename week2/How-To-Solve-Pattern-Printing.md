The Golden Rule

For every pattern:

1. Count rows
2. Count columns in each row
3. Find what should be printed at each position

Step 3: Find the formula

Stars decreasing:
4,3,2,1

Spaces decreasing:
3,2,1,0

Stars in pyramid:
1,3,5,7
= 2*row - 1





Most common formulas

Increasing stars     = row
Decreasing stars     = n - row + 1
Leading spaces       = n - row
Pyramid stars        = 2*row - 1
Reverse pyramid      = 2*(n-row)+1
