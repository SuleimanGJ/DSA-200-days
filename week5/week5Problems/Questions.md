# Java Basics Practice Problems

A structured collection of Java practice problems organized by difficulty level. Complete each level before moving to the next.

---

# 🟢 Level 1 – Very Easy (Java Basics)

## Objectives

* Variables
* Data Types
* Input/Output
* Arithmetic Operators

### Problems

1. Print "Hello, World!"
2. Print your name and age.
3. Add two numbers.
4. Swap two numbers using a third variable.
5. Swap two numbers without using a third variable.
6. Find the square of a number.
7. Find the cube of a number.
8. Convert Celsius to Fahrenheit.
9. Convert Fahrenheit to Celsius.
10. Find the area and perimeter of a rectangle.

# Recommended Learning Order

* ✅ Level 1 – Java Basics
* ✅ Level 2 – Input & Operators
* ✅ Level 3 – Conditional Statements
* ✅ Level 4 – Loops
* ✅ Level 5 – Loop Logic
* ✅ Begin Arrays
* ✅ Start Basic DSA
* ✅ Solve beginner competitive programming problems

## Goal

Complete **80–100 Java basics problems** before starting Arrays and Data Structures & Algorithms (DSA). A strong foundation in these topics will make advanced problem-solving significantly easier.
