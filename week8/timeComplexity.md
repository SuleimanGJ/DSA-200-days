# Time Complexity & Space
Without complexity analysis, you cannot compare algorithms 


## Time Complexity
Measures how the running time (not actual time coz it depend on a lot factor that can affects) grows as input size (N) grows.

It doesn't measure actual seconds


## Space Complexity
Measures extra memory used


## Big-O Cheat Sheet

O(1)  → HashMap Lookup  
O(log n) → Binary Search  
O(n)  → Linear Search  
O(n log n) → Merge Sort  
O(n²) → Bubble Sort  
O(2ⁿ) → Recursive Fibonacci, Backtracking  
O(n!) → Permutations  

## Rules to calculate time complexity
1 - Ignore constants  
2 - Keep dominant terms  
3 - Sequential loops add    
4 - Nested loops multiply
